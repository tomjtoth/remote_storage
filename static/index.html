About

This is almost like window.localStorage, except it is synced accross ALL clients of a certain webpage.

User manual

Simply insert the below tag in your webpage: 

<script src="https://ttj.hu:55589/remoteStorage.js"></script>
Then make calls as seen below:
  • let result = await remoteStorage.set(key, val) to store data
    • await could be omitted, unless you want to make sure you get the HTTP 200 back
  • let result = await remoteStorage.get(key) to retrieve data
  • let result = await remoteStorage.clear() to clear data that belongs to Origin
    • same as above, await could be omitted there, unless you want to make sure you get the HTTP 200 back
    • provided that CLEARING_SITE_DATA_ALLOWED=true is set in .env you get HTTP 200
    • otherwise you get HTTP 403
